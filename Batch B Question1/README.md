# JavaScript DOM & Event Handling

### Registration Form with Client-Side Validation

---

## Project Overview

This project is a Registration Form built using:

* HTML
* CSS
* JavaScript

The form performs client-side validation using JavaScript DOM manipulation and event handling.

Features:

* Name validation
* Email validation
* Password validation
* Confirm password matching
* Success message
* Confetti animation after successful registration

# Validation Rules

## Name

* Cannot be empty
* Only alphabets and spaces allowed

## Email

* Must follow proper email format

## Password

* Minimum 6 characters required

## Confirm Password

* Must match password field
